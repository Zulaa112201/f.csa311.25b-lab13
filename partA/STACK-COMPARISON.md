# Technology Stack Comparison

Энэхүү төслийн явцад бид хоёр өөр технологийг харьцуулж үзээд эцсийн сонголтоо хийсэн.

| Шалгуур | Java (Spring Boot) | Node.js (TypeScript) |
| :--- | :--- | :--- |
| **Хөгжүүлэлтийн хурд** | Дунд (Нүсэр тохиргоо) | Өндөр (Хурдан прототайп) |
| **AI Collaboration** | Сайн | Маш сайн (Context-д тохиромжтой) |
| **Robustness** | Маш сайн (Strong types) | Маш сайн (Static typing + Fast Fail) |
| **Орчин бүрдүүлэлт** | Төвөгтэй (JRE/Maven/IDE) | Хялбар (NPM/Vite/VS Code) |

**Эцсийн шийдвэр:** Бид AI-тай хамтран ажиллах уян хатан байдал болон хөгжүүлэлтийн хурдыг харгалзан **TypeScript** стакыг сонгосон.
