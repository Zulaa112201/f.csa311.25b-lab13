# AI Session 003: Unit Testing with Jest

**Асуудал:** `Jest` ашиглан тест бичих үед `@types/jest` танигдахгүй байх болон 10+ тест бичих шаардлагыг хангах.

**AI-ийн тусламж:**
- `package.json` болон `tsconfig.json` дээрх төрөл таниулах тохиргоонуудыг (types: ["jest"]) засаж өгсөн.
- Happy path болон Edge case-үүдийг хамарсан 10 тестийн загварыг гаргасан.

**Миний оруулсан хувь нэмэр:**
- Тестүүдийг нэг бүрчлэн ажиллуулж, `library.db` файл үүсэж байгаа эсэхийг шалгасан.
- "Target" хавтсыг устгаж, системийг цэвэрлэсэн.
