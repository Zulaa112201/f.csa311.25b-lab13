# AI Session Log 03: Loan Tracking Implementation

**Зорилго:** Ном олгох логик болон номын төлөвийг (availability) өөрчлөх функцийг хэрэгжүүлэх.

**AI-ийн оролцоо:**
- `Loan` entity-г `ManyToOne` харьцаатайгаар зохион байгуулах зөвлөмж өгсөн.
- `LoanService` дээр ном олгох үед `isAvailable` төлөвийг шалгах exception handle хийх логикийг санал болгосон.

**Үр дүн:**
- Ном олгох үед номын сангийн өгөгдлийн бүрэн бүтэн байдал хангагдаж байна.
