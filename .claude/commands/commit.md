# /push — Standardized Git Commit
Хийсэн ажлыг дүгнэж, дараах хэлбэрээр commit мессеж бич:
- `feat:` (шинэ боломж), `fix:` (алдаа зассан), `refactor:` (код зассан), `test:` (тест нэмсэн).
- Мессеж бүрийн төгсгөлд `Co-Authored-By: Claude <noreply@anthropic.com>` нэм.
- Одоогийн өөрчлөлтөд хамгийн тохиромжтой 3 өөр нэрийг санал болго.
