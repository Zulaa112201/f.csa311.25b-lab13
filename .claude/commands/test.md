# /factory — Test Driven Development (TDD)
Энэ кодод зориулж `Jest` тестүүдийг "Factory" зарчмаар үүсгэ:
- **Mocking:** `Repository`-г mock хийж, өгөгдлийн сангүйгээр ажиллах тест бич.
- **Async/Await:** Бүх асинхрон үйлдлүүдийг `resolves/rejects` ашиглаж шалга.
- **Coverage:** Кодны мөр бүрийг (Branch coverage) хамрах 5-10 тестийн кейс гарга.
